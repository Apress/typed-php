A Composer plugin which allows dependencies to hook themselves into PHP config files.

## Installation

```sh
❯ composer require "typedphp/composer-hook-plugin:*"
```

## Testing

```sh
❯ composer create-project "typedphp/composer-hook-plugin:*" .
❯ vendor/bin/phpunit
```
